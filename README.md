🐍 Snake Game – Unity
🎮 Overview

This project is a recreation of the classic Snake Game built using the Unity Game Engine and C#. The player controls a snake that moves around the screen to collect food. Each time the snake eats food, it grows longer and the player's score increases. The game ends if the snake collides with the wall or its own body.

This project demonstrates basic game development concepts such as player movement, collision detection, object spawning, and score management.

✨ Features
Smooth snake movement
Food spawning system
Snake grows when food is eaten
Score tracking
Game over on collision with wall or snake body
Simple and interactive gameplay

🛠️ Technologies Used
Unity Engine
C#
Unity Physics & Collision System
